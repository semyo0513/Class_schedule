# 학교 통합 시간표 관리 시스템 (School Timetable Manager)

## 1. 프로젝트 개요

### 프로젝트명

School Timetable Manager (STM)

### 목적

중·고등학교 교사가 시간표를 효율적으로 관리하고, 수업 교체·대강·행사 운영·특별실 관리 등을 통합적으로 처리할 수 있는 웹 기반 시간표 관리 플랫폼 구축

### 기술 스택

| 영역          | 기술                                |
| ----------- | --------------------------------- |
| Frontend    | HTML, CSS, JavaScript, React (선택) |
| Hosting     | GitHub Pages                      |
| Backend API | Google Apps Script (GAS)          |
| Database    | Google Sheets                     |
| 인증          | Google OAuth                      |
| 파일 처리       | SheetJS(xlsx)                     |
| PDF 출력      | jsPDF                             |
| 배포          | GitHub Actions                    |
